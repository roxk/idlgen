# Contributing to idlgen

## Prerequisite

### Core
- dotnet 6+

### Sample Apps
- Visual Studio 2022/2026
- C++ Workload
- WinUI Workload (optional)
- nuget (the cli tool)

## Getting Started

1. Install dotnet 6+ SDK
2. Run the following
```
dotnet run ./scripts/getWinLibs.cs
dotnet run ./scripts/test.cs
```
